## Overview

Vulcan uses SQLMesh's virtual data environments to enable safe, isolated development and testing without duplicating data or incurring additional warehouse costs.

### Key Concepts

- **Virtual Environments**: Isolated views of your data transformations
- **Plan and Apply**: Review changes before applying them
- **Environment Management**: Dev, staging, and production workflows
- **Scheduling**: Integration with Airflow, cron, and other orchestrators
