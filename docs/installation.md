# Installation

This page provides instructions for installing Vulcan on your computer.

## Python virtual environment

It is recommended, but not required, that you use a python virtual environment with Vulcan.

First, create the virtual environment:
```bash
python -m venv .venv
```

Then activate it:
```bash
source .venv/bin/activate
```

## Install Vulcan core

Install the core Vulcan library with `pip`:
```bash
pip install vulcan
```
